Instructions to execute the code:
- Install XAMPP server and connect to the server
- Import the sql code into phpmyadmin.
- Set Servername: localhost, Username: root, Password:,
- Database name: OnlineDeliverySystem so that the connection works fine.
- Save the project folder in Applications/XAMPP/xamppfiles/htdocs and name it as OnlineDeliverySystem
- Run http://localhost/OnlineDeliverySystem/code/home_page.html in your browser.
